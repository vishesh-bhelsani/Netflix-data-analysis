# 🎬 Netflix Data Analysis

## About
Analysis of 8,807 Netflix titles using Python, Pandas and Matplotlib.

## Dataset
- 8,807 Netflix movies and TV shows
- 12 columns of information

## Key Findings
1. Netflix has more Movies than TV Shows
2. USA produces the most content on Netflix
3. Dramas and Comedies are the most popular genres
4. Netflix grew rapidly between 2015-2019
5. Most content is for mature audiences (TV-MA)

## Tools Used
- Python
- Pandas
- Matplotlib
